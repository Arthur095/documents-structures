Les programmes suivants doivent êtres installés pour faire fonctionner le script "generate.sh" : Convertit directement les odt présents dans transformation/input au format TEI dans le répertoire output et les valide.

unzip;
java 8; (ou version plus récente)
xmllint; (libxml2)

Vous pouvez coller la ligne suivante dans l'invite de commandes :

apt-get install unzip; apt-get install java; apt-get install libxml2;
